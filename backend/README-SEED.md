# 数据库种子数据说明

## 运行种子脚本

已修复学生和教师列表的500错误问题,现在可以正常使用新增和编辑功能。

### 添加测试数据

在backend目录下运行以下命令来添加测试数据:

```bash
npm run seed
```

或者直接运行:

```bash
npx ts-node prisma/seed.ts
```
