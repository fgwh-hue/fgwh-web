## 常见问题

### Q: 添加学生时提示"学号不能为空"
A: 确保填写了学号字段

### Q: 添加后列表仍然为空
A:
1. 检查浏览器控制台是否有错误
2. 检查后端日志是否有错误
3. 刷新页面重新加载数据

### Q: 提示401未授权
A:
1. 重新登录
2. 清除浏览器缓存和localStorage
3. 使用正确的账号密码

## 后端API测试

可以使用Swagger文档测试API:
http://localhost:3001/api/docs

或使用curl命令测试(需要先获取token):
```bash
# 登录获取token
curl -X POST http://localhost:3001/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@example.com","password":"password"}'

# 添加学生(替换YOUR_TOKEN)
curl -X POST http://localhost:3001/api/students \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "studentNo": "S001",
    "studentName": "测试学生",
    "studentGender": 1,
    "studentMajor": "计算机科学与技术",
    "studentClass": "计科1班"
  }'
```
