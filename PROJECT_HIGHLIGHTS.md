# 项目亮点

## 🌟 项目概述

本项目是一个基于Vue 3、Vite 7、TypeScript和Naive UI的现代化AI工具平台，集成了教育管理系统和AI助手功能，为用户提供功能丰富、体验优秀的在线学习和教学平台。

---

## 🚀 技术栈亮点

### 前端核心技术
- **Vue 3.5.21**：采用最新的Vue 3 Composition API，提供更好的代码组织和复用性
- **Vite 7.1.5**：极速的开发服务器和构建工具，秒级冷启动，即时热更新
- **TypeScript 5.9.2**：完整的类型安全，减少运行时错误，提升代码质量
- **Naive UI 2.43.1**：现代化的UI组件库，提供丰富的组件和主题定制能力

### 样式与工具
- **UnoCSS 66.5.1**：原子化CSS引擎，提供灵活的样式定制和优秀的性能表现
- **Pro Naive UI 3.1.1**：基于Naive UI的高级组件封装，提供开箱即用的业务组件
- **Tailwind Merge 3.3.1**：智能的Tailwind CSS类名合并工具

### 状态管理与路由
- **Pinia 3.0.3**：Vue 3官方推荐的状态管理库，提供响应式的状态管理
- **Vue Router 4.5.1**：官方路由管理器，支持动态路由和路由守卫
- **@elegant-router/vue 0.3.8**：优雅的路由解决方案，自动生成路由和类型定义
