# 项目亮点

## 🌟 项目概述

本项目是一个基于**Vue 3 + NestJS**的现代化AI学习教育系统，采用前后端分离架构，前端使用Vue 3、Vite构建，后端使用NestJS提供API服务，通过Prisma ORM连接PostgreSQL数据库，实现了完整的用户认证、权限管理和教育管理功能。

---

## 🚀 技术栈亮点

### 前端核心技术
- **Vue 3.5+**：采用最新的Vue 3 Composition API，提供更好的代码组织和复用性
- **Vite 5.0+**：极速的开发服务器和构建工具，秒级冷启动，即时热更新
- **TypeScript 5.0+**：完整的类型安全，减少运行时错误，提升代码质量
- **Naive UI 2.40+**：现代化的UI组件库，提供丰富的组件和主题定制能力

### 前端样式与工具
- **Tailwind CSS**：原子化CSS框架，提供灵活的样式定制和优秀的性能表现
- **Pinia 3.0+**：Vue 3官方推荐的状态管理库，提供响应式的状态管理
- **Vue Router 4.0+**：官方路由管理器，支持动态路由和路由守卫

### 后端核心技术
- **NestJS 10.0+**：现代化的Node.js后端框架，提供依赖注入、模块化架构
- **Prisma ORM**：强大的ORM工具，简化数据库操作，提供类型安全的数据访问
- **JWT认证**：安全的用户认证和授权机制，支持Token刷新
- **PostgreSQL**：强大的关系型数据库，支持复杂查询和数据完整性

### API开发
- **Swagger/OpenAPI**：自动生成API文档，方便前后端协作
- **Class Validator**：强大的数据验证，确保API输入安全
- **bcrypt**：密码加密存储，保障用户数据安全
